# Included demo data

The files in this directory contain 60 public sample records collected from
[Books to Scrape](https://books.toscrape.com/), a site created for scraping
practice.

They are included only so the Streamlit dashboard and Docker image have a
useful read-only demo immediately after startup. Run `python run_pipeline.py`
to collect a fresh dataset into the parent `data/` directory.
