## Title

Python Product Data Scraper with Excel Reporting and Dashboard

## Short description

I built a repeatable product-data workflow that collects catalog and detail
records, validates and deduplicates them, stores the results in SQLite, and
creates a formatted Excel report. A Streamlit dashboard makes it easy to search,
filter, compare, and download the collected records.

The demo uses a website intended for scraping practice. The same structure can
be adapted to permitted public catalogs after checking the site's terms and
rate limits.
